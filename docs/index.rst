Email Hippo WHOIS API Documentation
***********************************
`Email Hippo`_ \'WHOIS\' :term:`API` services facilitate easy, fast and scalable access to the global :term:`WHOIS` databases in both structured and un-structured formats.

This document will show you how to get up and running with the the service. 
You will have the basics of the :term:`API` up and running in 15 minutes or less.
